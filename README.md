# Bismart Backend

## Get Started

1. `$ npm install`
2. Launch Enviornment:
    * `$ node app.js or nodemon app.js`
3. In Cluster mode with the help of pm2 [optional step]:
    * `$ pm2 start app.js or pm2 start app.js -i <no of instances>`
4. Open in browser:
    * open `http://localhost:9890`
